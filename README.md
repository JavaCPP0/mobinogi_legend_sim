# 전설 패션 승급 시뮬레이터

게임 내 의상 승급(엘리트 → 에픽 → 전설)을 확률적으로 시뮬레이션하는 파이썬 스크립트입니다.

머리 / 상의 / 하의 / 장갑 4부위에 대해,
사용자가 입력한 보유 에픽·엘리트 수량만을 사용하여

* 각 부위 전설 1개씩을 모두 완성할 수 있는지(성공/실패)
* 그 과정에서 사용된 현금
* 부위별 에픽/전설 승급 도전 횟수

를 여러 번 시뮬레이션해 통계를 확인할 수 있습니다.

---

## 승급 규칙

스크립트에서 사용하는 규칙은 다음과 같습니다.

등급 구조

* 엘리트 → 에픽 → 전설

확률 및 비용

* 엘리트 3개 → 에픽 승급 도전

  * 성공 확률: 25%
  * 시도당 비용: 500원
  * 성공: 엘리트 3개 소모, 에픽 1개 생성
  * 실패: 엘리트 2개만 소모, 엘리트 1개는 남음

* 에픽 3개 → 전설 승급 도전

  * 성공 확률: 20%
  * 시도당 비용: 3,000원
  * 성공: 에픽 3개 소모, 전설 1개 생성
  * 실패: 에픽 2개만 소모, 에픽 1개는 남음

기타 규칙

* 부위: 머리 / 상의 / 하의 / 장갑 (4부위)
* 각 부위는 자기 부위 아이템으로만 승급 (부위 간 재료 공유 없음)
* 엘리트는 “입력한 수량까지만” 사용 가능 (추가 구매 없음)
* 목표: 4부위 모두 전설 1개씩 보유한 상태

시뮬레이션 종료 조건

* 성공: 4부위 모두 전설 1개씩 완성
* 실패: 어떤 부위도 더 이상

  * 에픽 3개 이상 보유
  * 엘리트 3개 이상 보유
    둘 다 만족하지 못해 승급 도전을 할 수 없는데, 아직 전설 4부위가 아닌 경우

## 사용 방법

실행 후, 프로그램이 순서대로 질문을 합니다.

1. 각 부위별 현재 보유 에픽/엘리트 개수 입력

예시:

```text
==== 전설 패션 시뮬레이터 (현금 기준) ====
각 부위별 현재 보유한 '에픽'과 '엘리트' 개수를 입력해주세요.
목표: 머리/상의/하의/장갑 전설 1개씩 만들기
엘리트는 입력하신 수량까지만 사용할 수 있고, 떨어지면 더 이상 합성을 못 합니다.
실패 시 재료 1개는 남는 규칙이 적용되어 있습니다.

머리 에픽 개수: 0
머리 엘리트 개수: 50
상의 에픽 개수: 0
상의 엘리트 개수: 40
하의 에픽 개수: 0
하의 엘리트 개수: 30
장갑 에픽 개수: 1
장갑 엘리트 개수: 40
```

2. 시뮬레이션 반복 횟수 입력

```text
시뮬레이션 횟수 (기본값 30, 그냥 Enter): 30
```

* 그냥 Enter → 30회
* 숫자 입력 → 해당 횟수만큼 시뮬레이션 반복

## 출력 형식

각 시뮬레이션(1회차, 2회차, …)에 대해 다음과 같이 출력됩니다.

예시:

```text
1회차:
머리: 에픽승급도전횟수:9, 전설승급도전횟수:1 실패
상의: 에픽승급도전횟수:7, 전설승급도전횟수:0 실패
하의: 에픽승급도전횟수:5, 전설승급도전횟수:0 실패
장갑: 에픽승급도전횟수:6, 전설승급도전횟수:1 성공
사용된 현금: 27,000원
```

설명

* 각 부위별로

  * 에픽승급도전횟수: 엘리트 → 에픽 승급 시도 횟수
  * 전설승급도전횟수: 에픽 → 전설 승급 시도 횟수
  * 성공/실패: 해당 부위가 전설 1개를 달성했는지 여부
* 사용된 현금: 해당 회차에서 4부위 승급 전체에 들어간 총 비용

마지막에는 전체 요약이 출력됩니다.

예시:

```text
===== 요약 =====
총 시뮬레이션 횟수: 30회
성공: 3회, 실패: 27회

[성공한 경우 기준 현금 사용량]
  평균: 35,500원
  최소: 24,500원
  최대: 47,000원
```

설명

* 성공: 4부위 모두 전설 1개씩 완성된 회차 수
* 실패: 그 외(엘리트/에픽이 다 떨어져 더 이상 도전 못 하는 상태에서 목표 미달)
* 아래 통계는 “성공한 회차들만” 기준으로

  * 평균/최소/최대 현금 사용량을 보여줍니다.

## 설정 변경 방법

스크립트 상단의 상수들을 수정하여, 게임 내 실제 값에 맞게 조정할 수 있습니다.

```python
PROB_ELITE_TO_EPIC = 0.25   # 엘리트 3개 -> 에픽 성공 확률
PROB_EPIC_TO_LEGEND = 0.20  # 에픽 3개 -> 전설 성공 확률

COST_ELITE_TO_EPIC = 500    # 엘리트 승급 1회당 현금
COST_EPIC_TO_LEGEND = 3000  # 전설 승급 1회당 현금
```

예를 들어

* 에픽 승급 확률이 30%라면 `PROB_ELITE_TO_EPIC = 0.3`
* 전설 도전 비용이 5,000원이라면 `COST_EPIC_TO_LEGEND = 5000`
  처럼 수정 후 다시 실행하시면 됩니다.
